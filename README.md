# 💸 Expense Tracker (C++)

A simple command-line based Expense Tracker built using C++ to manage daily expenses efficiently.

## 🚀 Features

* Add and store expenses with category
* View all transactions
* Track total spending
* Basic analytics (category-wise spending)
* File handling for data persistence

## 🛠️ Tech Stack

* C++
* File Handling (fstream)
* STL (Vectors, Strings)

## 📌 Use Case

Designed for students to track and manage monthly expenses easily.
